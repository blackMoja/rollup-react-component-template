# [Color System](https://www.figma.com/file/4iVAWypeGjx1xE0Rbjg2Sa/Soomgo-Brand-Foundation-v.2.0?node-id=11%3A237)
Main Color (500) 의 기준으로 추출한 컬러 팔레트입니다. 숨고 프로덕트 디자인 시 아래의 팔레트 내에서 디자인을 지향합니다.<br/>
체크 표시된 컬러 팔레트는 프로덕트 내에서 사용 빈도가 높은 팔레트로 개발에서 50 ~ 900 컬러 모두 등록이 필요합니다. 이 외 컬러는 Semantic 컬러 내 사용을 지향합니다.

## Basic Color

### Teal ✅ 
브랜드 대표 컬러, 링크 및 포인트로 사용
> ex) Teal 500 - CTA Button<br />
>	Teal 900 - CTA Button Pressed Status

<div className="d-flex justify-center py-1 my-4">
  {Object.entries(parsedBasicColors.teal).sort(([vA], [vB]) => vA - vB).map(([variant, colorString], key) => (
    <div key={key} className={`color small${['100', '500', '900'].includes(variant) ? ' circle' : ''} sg-bg-color-teal-${variant}`}>
      <Typography variant="interface-detail-bold">{variant}</Typography>
      <Typography variant="interface-detail-bold">{colorString}</Typography>
    </div>
  ))}
</div>

### Teal Gradation

> ex) 프로필 영역, Chat

<div className="d-flex d-col py-1 my-4">
